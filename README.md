# vue-rabbit

小兔鲜儿电商项目（vue3全家桶）
## 技术栈

Vue3全家桶：create-vue、Pinia、ElementPlus、Vue3+Setup、Vue-Router、VueUse
## 项目亮点

<ul>
<li>基于业务逻辑的组件拆分思想</li>
<li>长页面吸顶交互实现</li>
<li>SKU电商组件封装</li>
<li>图片懒加载指令封装</li>
<li>通用逻辑函数封装</li>
<li>面板插槽组件等业务通用组件封装</li>
<li>路由缓存问题处理</li>
<li>支付宝第三方支付</li>
</ul>


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
